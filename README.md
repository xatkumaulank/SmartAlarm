# SmartAlarm
Ứng dụng báo thức phiền toái
### Tính năng chính
- Bạn phải giải các bài toán đơn giản và kích thích trí não của bạn
- Điền phần còn thiếu vào chỗ trống
- Hoặc trả lời những câu hỏi về kiến thức xã hội, đời sống, ...
